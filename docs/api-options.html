Опции

Список, включающий все варианты настраиваемых опций

parent

Тип: String
Default: 'body'

Родительский элемент в который вливается слайдер

photos

Type: Array
Default: ['']
Допустимые значения: ['folder/nameImg.format','']

Путь к каждой картинке (содержимое слайдера)

Заполняет атрибут src.

btnNav

Type: Array
Default: ['']
Допустимые значения: ['.nameEl','#nameEl']

Идентификаторы кнопок отвечающих за перелистывание

dots

Type: Boolean
Default: true

Наличие у слайдера точек идентификаторов

btnPlay

Type: Array
Default: ['']
Допустимые значения: ['.nameEl','#nameEl']

Идентификаторы кнопок отвечающих за автопроигрывание

btnStartEnd

Type: Array
Default: ['']
Допустимые значения: ['.nameEl','#nameEl']

Идентификаторы кнопок перелистывающие на первый или последний слайд

current

Type: Number
Default: 0

Индекс начальной картинки слайдера

autoplay

Type: Boolean
Default: true

Наличие у слайдера функций и кнопок проигрывателя
